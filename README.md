# ImageSegmentator
## Description
Image Segmentation using K-Means

## Requires
- matplotlib

## Quickstart
```bash
git clone https://github.com/azhu7/ImageSegmentator.git
cd ImageSegmentator/src
python segment.py ../sample_pictures/owl.jpg
```
